# Raspberry Pi Setup
## Install on Pi:
### apt
```
sudo apt update
```
```
sudo apt upgrade
```
```
sudo apt install git
```
```
sudo apt install pyhton3-pip
```
```
sudo apt install pyhton3-venv
```
### to run acme on bare metal
This command in the acme folder:
```
python3 -m venv .
```
Then install requirements:
```
./bin/pip3 install -r requirements.txt
```
